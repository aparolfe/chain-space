How do I use chain-space?

Enter your own pattern and hit the "Create Chart" button! Here are the rules for the pattern text...

  • Chain-space recognizes instructions in US terminology. Currently supported stitches are ch, sc, hdc and dc.
  • Start each line with "Row" followed by a number. Each row should have a distinct number.
  • Chain-space assumes that every stitch in the previous row will be worked into. To leave a stitch unworked, use "skip" or "sk". You can skip multiple stitches by adding a number to the end, for example "sk2" or "skip 5".
  • If you have multiple stitches of the same type made into consecutive stitches of the previous row, you can add that number to the end of the stitch name, like "ch9" or "sc 3".
  • For multiple stitches worked in the same stitch, enclose all the stitches in parenthesis. Eg: "(dc5)" will make a 5-dc fan in the next stitch.
  • If a row is the same as a previous row, you can use "Repeat" or "Rep" to repeat that row, for example "Row 3: Repeat Row 2".
